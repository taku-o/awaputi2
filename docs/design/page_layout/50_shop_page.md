# ショップページレイアウト

## 表示項目

### ヘッダー情報
- タイトル「アイテムショップ」
- ユーザー名
- 所持AP
- 総TAP

### アイテムリスト
- スコア倍率アップ
- 復活
- レア率アップ
- HP増加
- コンボ強化
- アイテム効果リセット

### アイテム情報
- アイテム名
- 説明文
- 現在のレベル
- 次のレベルでの効果
- 購入コスト
- 最大レベル

### アイテム状態表示
- 購入可能（青色背景）
- 購入不可（オレンジ色背景）
- 最大レベル到達（グレーアウト）

### 操作説明
- キーボード操作（↑↓キー）
- マウス操作（クリック）
- スクロール機能

### 操作要素
- アイテム選択
- アイテム詳細表示
- 購入実行
- メニューに戻る

### 表示仕様
- 最大表示アイテム数: 6個
- アイテムカードサイズ: 幅 canvas.width-40px、高さ 80px
- スクロール対応
