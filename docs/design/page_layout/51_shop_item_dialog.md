# ショップアイテムダイアログレイアウト

### アイテム基本情報
- アイテム名
- アイテムアイコン
- 説明文
- カテゴリ

### レベル情報
- 現在のレベル
- 最大レベル
- 現在の効果
- 次のレベルでの効果

### コスト情報
- 現在の購入コスト
- レベル別コスト一覧
- 所持APとの比較

### 効果詳細
- 効果の仕組み
- 適用タイミング

- 使用例

### 購入確認
- 購入ボタン
- キャンセルボタン
- 購入後の状態表示

- 購入実行
- ダイアログ閉じる
- 詳細情報表示
- ヘルプ表示
